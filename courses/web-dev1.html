Web Development I - HTML & CSS

Course Overview

This beginner-level course introduces students to the essentials of web design and development, focusing on frontend creation with HTML and CSS. Students will learn about the infrastructure of the Internet, how to build and publish web pages, and employ various layout and styling techniques.

Course Objectives

  • Understand Internet infrastructure and the basics of web technologies.
  • Become familiar with HTML and CSS.
  • Learn web page layout techniques, including the use of div tags, lists, and the concepts of absolute vs. relative positioning.
  • Understand and apply advanced layout strategies using CSS Flexbox and Grid to create responsive designs.
  • Implement multimedia elements into web pages, such as images, videos, and favicons, enhancing user engagement.
  • Develop skills in search engine optimization (SEO) and web accessibility to maximize website visibility and usability.

"Web Development" vs. "Web Design"

A web designer focuses on the visual aspects of a website: layout, colors, fonts, and overall appearance. They create mockups and make sure the site is visually appealing and user-friendly, and they work with tools like Photoshop or design frameworks.

A web developer, on the other hand, focuses on the technical side: building the website's structure using code like HTML, CSS, and JavaScript. Developers take the designer's mockup and turn it into a functional, working website.

In this course, you'll be learning web development: how to write code that makes websites work and look good using CSS styling tools. While we'll cover how to style a site, we won't get into graphic design principles. My focus is on teaching you how to build and style functional web pages, so don't worry if you don't consider yourself an artist!
