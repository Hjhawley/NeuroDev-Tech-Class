Introduction to Linux

Course Overview

This course introduces students to the Linux operating system through practical, beginner-friendly tasks. Designed for learners with basic computer literacy, it emphasizes understanding the Linux ecosystem, navigating the terminal, managing files and processes, and using open-source tools. By the end of the course, students will have the skills to use Linux comfortably as a daily OS or development environment.

Course Objectives

  • Understand what Linux is and how it differs from Windows/macOS.
  • Use the command line to navigate and manipulate the file system.
  • Manage software using package managers.
  • Understand file permissions and user management basics.
  • Perform basic system monitoring and troubleshooting.
  • Gain confidence with a Linux desktop environment.
